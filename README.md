# Recreação da Página de Vídeo do YouTube

Este projeto foi desenvolvido como parte do **Bootcamp Front-End Web da DIO (Digital Innovation One)**. O objetivo principal foi recriar a interface de uma página de reprodução de vídeos do YouTube, focando na fidelidade do layout e nas boas práticas de desenvolvimento web.

## 🎯 Objetivo do Projeto

O foco central deste desafio foi **treinar o uso de Flexbox** para a construção de layouts complexos e responsivos. Durante o desenvolvimento, foram abordados conceitos essenciais como:
- Alinhamento de elementos em eixos verticais e horizontais.
- Distribuição de espaçamentos dinâmicos (`gap`, `justify-content`, `align-items`).
- Resolução de comportamentos do Flexbox, como o controle de encolhimento de componentes estruturais (`flex-shrink`).
- Estruturação de "cards" reutilizáveis para a barra lateral de vídeos recomendados.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Para a estruturação semântica da página (uso correto de tags como `<main>`, `<aside>`, `<nav>`, entre outras).
- **CSS3:** Para a estilização, definição de cores, tipografia e toda a mecânica de posicionamento com **Flexbox**.
